# DevOps-ассистент автоматизирующий некоторые рутинные задачи

---
## check-env
**Использование:** ./check-env
Выводит версии установленного ПО (git, docker, dotnet, curl, python), если не установлены, сообщает об этом

---
## create-project-structure
**использование:** ./create-project-structure <name>
Создаёт директорию <name> со структурой каталогов, .gitignore и инициализирует git
